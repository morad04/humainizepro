# HUMAINIZE

AI Pattern Remover — Detects and removes 24 AI writing patterns to bypass AI detectors.

## Features
- Real-time AI pattern detection (24 patterns)
- 8-pass humanization engine targeting perplexity, burstiness & structure
- Strength slider: Light / Medium / Aggressive
- Premium dark glassmorphism UI
- 100% client-side, no API keys needed

## Usage
Open `index.html` in any browser or deploy to any static hosting.
